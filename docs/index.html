React Native Audio Pro

A React Native module for robust audio playback from remote URLs. Ideal for audiobook and podcast apps.

Overview

React Native Audio Pro provides modern audio playback features for React Native apps. Stream remote files, control audio in the background, and access lock screen controls easily on both Android and iOS.

Installation

npm install react-native-audio-pro
# or
yarn add react-native-audio-pro

Requirements

  • React Native 0.60 or higher
  • iOS 15.0 or higher
  • Android 13 (API Level 33) or higher

Platform Setup

iOS

  • Enable Background Modes > Audio, AirPlay, and Picture in Picture
  • Set iOS Deployment Target to 15.0+

Android

// android/build.gradle
buildscript {
  ext {
    minSdkVersion = 33
    compileSdkVersion = 33
    targetSdkVersion = 33
  }
}
<!-- AndroidManifest.xml -->
<uses-permission android:name="android.permission.FOREGROUND_SERVICE" />
<uses-permission android:name="android.permission.WAKE_LOCK" />

API Overview

Methods

  • play(track)
  • pause()
  • resume()
  • stop()
  • seekTo(ms)
  • seekForward(seconds?)
  • seekBack(seconds?)

Events

  • addStateListener(callback)
  • addNoticeListener(callback)

Track Type

type AudioProTrack = {
  url: string;
  title: string;
  artwork: string;
  album?: string;
  artist?: string;
}

Basic Usage

import AudioPro from 'react-native-audio-pro';

const track = {
  url: 'https://example.com/audio.mp3',
  title: 'Example Track',
  artwork: 'https://example.com/art.jpg',
  artist: 'Artist Name'
};

AudioPro.play(track);

AudioPro.addStateListener((state) => {
  console.log('State:', state);
});

AudioPro.addNoticeListener((notice) => {
  console.log('Notice:', notice);
});
