React Native Audio Pro

A React Native module for robust audio playback from remote URLs, live streams, and local files. Ideal for music, audiobook, podcast, and streaming apps. Works on iOS, Android, and Web.

🚀 Features

  • 🎵 Powered by Media3 (Android) & AVFoundation (iOS)
  • 🔒 Lock screen controls & background playback
  • 🔔 Automatic handling of notification center (Android & iOS)
  • 🌐 Web support via HTML5 Audio API
  • 📡 Support for static remote files, live streams (HLS, DASH, RTSP, RTMP), and local files (for both audio and artwork)
  • 📦 Built fully in TypeScript – no JS wrappers
  • 🧠 Simple React hook & module interface
  • ⚡️ Lightweight, fast, and production-ready

🎧 React Hook in Action

import { useAudioPro } from 'react-native-audio-pro';

export default function Player() {
  const { state, duration, position } = useAudioPro();
  return (
    <View>
      <Text>State: {state}</Text>
      <Text>Position: {position} / {duration}</Text>
    </View>
  );
}

🧩 Module in Action

import {AudioPro} from 'react-native-audio-pro';

AudioPro.configure({ contentType: 'speech' });

// Works with static remote files, live streams, and local files
const track = {
  id: 'track-001',
  url: 'https://example.com/audio.mp3', // Remote file: 'https://example.com/livestream' or local file
  title: 'Sample Track',
  artwork: 'https://example.com/image.jpg', // Remote image or local image
  artist: 'Narrator Name',
};

// Play with default options
AudioPro.play(track);

// With custom options
AudioPro.play(track, {
  autoPlay: false,
  headers: {
    audio: { 'Authorization': 'Bearer token123' },
    artwork: { 'Cache-Control': 'max-age=3600' }
  }
});
